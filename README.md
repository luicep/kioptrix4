## Presentation Overview

The presentation is the primary project deliverable. It communicates the complete assessment narrative from initial reconnaissance through effective Linux root access, while connecting every significant technical claim to supporting evidence in this repository.

### What the presentation covers

- Authorized scope and isolated-lab methodology
- Network and service discovery
- Login-request mapping and SQL injection validation
- Credential extraction and authenticated SSH access
- Restricted-shell escape and privilege baselining
- Application source-code root-cause analysis
- MySQL misconfiguration and root-level command execution
- Comparable real-world incidents and documented business impact
- Prioritized remediation and post-remediation validation
